# **Cricket Scoreboard** (Assembly Language)

A console-based cricket scoreboard system developed using Assembly Language (MASM) with the Irvine32 library. This project simulates a real-time cricket match, allowing users to manage scoring, overs, wickets, and match flow interactively.

# **Project Description**

The Cricket Scoreboard System is designed to replicate the functionality of a real cricket match scoreboard in a console environment. It provides complete control over match operations including batting, bowling, scoring, and innings management.

The system is built using low-level programming concepts, demonstrating efficient use of Assembly Language for handling logic, user input, and real-time updates.


# **Features**
Toss functionality (decide batting/bowling)
Real-time score updates
Runs management (0, 1, 2, 3, 4, 6)
Wicket handling
Over tracking system
Extras handling (No Ball, Wide, etc.)
Innings switching
Match result calculation
Console-based interactive interface
Technologies Used
Assembly Language (MASM)
Irvine32 Library
Microsoft Visual Studio / MASM Assembler

# **How to Run**
Install MASM (Microsoft Macro Assembler)
Setup Irvine32 Library
Open project in Visual Studio or MASM environment
Compile the .asm file
Run the program in console
